# 傲虎走访 · 访谈记录卡

现场走访用的移动端访谈记录卡（纯静态 HTML + JS，无构建步骤）。手机浏览器直接打开即可：

- 按角色（主管 / 业务员 / 采购员…）逐题访谈，随手「记一笔」；
- 数据与系统盘点清单现场打勾；
- 试点适配度六维评分，当场出结论；
- 一键导出 .md / 复制到剪贴板带走。

所有记录存在本机浏览器 localStorage，不上传任何服务器。

## 文件

| 文件 | 作用 |
|---|---|
| `index.html` | 页面（结构 + 样式） |
| `visit-data.js` | 题库 / 盘点清单 / 评分维度数据 |
| `visit-app.js` | 交互、localStorage、导出逻辑 |
